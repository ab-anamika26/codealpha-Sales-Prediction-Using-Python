# 📈 Sales Prediction Using Python

## Project Overview

This project predicts product sales based on advertising expenditure using Machine Learning.

The dataset contains advertising budgets for:

- TV
- Radio
- Newspaper

The objective is to predict future sales using Linear Regression and analyze how different advertising platforms influence sales.

---

## Dataset

Source:
https://www.kaggle.com/datasets/bumba5341/advertisingcsv

Dataset Features:

| Feature | Description |
|----------|-------------|
| TV | TV advertising budget |
| Radio | Radio advertising budget |
| Newspaper | Newspaper advertising budget |
| Sales | Product sales |

## Business Insights

- TV advertising has the strongest influence on sales.
- Radio advertising also contributes significantly.
- Newspaper advertising has the least impact.
- Increasing TV and Radio advertising budgets is likely to generate better sales.
- The model helps estimate future sales for planned advertising budgets.

---

## Future Prediction

The original dataset does not contain time information.

To visualize future sales, hypothetical advertising budgets were created for the next 12 months, and the trained Linear Regression model predicted the expected sales.
